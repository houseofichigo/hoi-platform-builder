# House of Ichigo logos

Logo PNGs to be added — use `logo-navy-mark.png` on light surfaces and `logo-white-mark.png` on dark surfaces.

Available assets from the design system (to be dropped into this folder):

- `logo-navy-mark.png` — navy brushstroke 会 in navy circle (light backgrounds, default for this app)
- `logo-white-mark.png` — white brushstroke 会 in white circle (dark backgrounds)
- `logo-terracotta-mark.png` — AVOID per latest design direction
- `lockup-navy.svg`, `lockup-terracotta-light.svg`, `lockup-white-dark.svg` — full logo with wordmark

Never render the 会 character via CSS or web font. Always use these PNGs.
